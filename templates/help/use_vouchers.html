Save money on your FarmHub orders by using discount codes and vouchers. Here's everything you need to know about applying discounts.

Where to Get Discount Codes

FarmHub discount codes may be available from:

  • Promotional campaigns: Special events and seasonal sales
  • New user promotions: Welcome discounts for first-time buyers
  • Shop promotions: Individual sellers may offer discounts
  • Social media: Follow FarmHub for exclusive codes
  • Email newsletters: Subscribe to receive special offers

How to Apply a Discount Code

Step 1: Add items to your cart and proceed to checkout.
Step 2: Look for a "Discount Code", "Promo Code", or "Voucher" field on the checkout page.
Step 3: Enter your discount code exactly as provided (codes are usually case-sensitive).
Step 4: Click "Apply" or "Use" to apply the discount.
Step 5: Verify that the discount has been applied to your order total.
Step 6: Complete your order with the discounted price.
Copy and paste discount codes to avoid typos. Even one wrong character will prevent the code from working.

Common Voucher Terms

  • Minimum purchase: Some codes require a minimum order amount
  • Expiration date: Codes may only be valid for a limited time
  • Single use: Most codes can only be used once
  • Product restrictions: Some codes may only apply to certain products
  • New users only: Some promotions are for first-time buyers only
Most discount codes cannot be combined with other offers. Only one code can typically be applied per order.

Troubleshooting Discount Codes

Code Not Working?

  • Check expiration: The code may have expired
  • Verify spelling: Make sure you entered it correctly
  • Check requirements: You may not meet minimum purchase requirements
  • Already used: Single-use codes can't be reused
  • Account restrictions: Some codes are for specific users

Discount Not Showing?

  • Refresh the checkout page
  • Remove and re-add the code
  • Check if the code applies to your selected items
If you have a valid code that isn't working, contact customer support with the code details and your order information.
